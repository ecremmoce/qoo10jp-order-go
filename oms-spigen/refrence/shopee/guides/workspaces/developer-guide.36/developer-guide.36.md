# 서비스 약관

## 이용 약관 - 서비스 약관

**발효일: 2023년 4월 13일**

본 서비스 약관("본 서비스 약관")은 귀하가 Stripe에서 제공하는 제품(아래 정의됨)에 액세스하고 사용하는 데 적용되는 약관을 설명합니다. "제품"은 귀하가 API 서비스(아래 정의됨)를 사용할 수 있도록 플랫폼을 통해 제공되는 다양한 기능, 콘텐츠, 서비스 및 기능을 의미합니다. 이러한 제품은 Stripe, Inc. 및 그 계열사("Stripe", "당사", "저희")에서 제공합니다.

제품에 액세스하거나 사용함으로써 귀하는 Stripe와 구속력 있는 계약을 체결하는 것입니다. 귀하는 본 서비스 약관에 포함된 약관을 읽고 이해했으며 이에 동의하는 것을 인정합니다. 본 서비스 약관에는 Stripe 플랫폼 개인정보처리방침 및 쿠키 정책과 API 서비스 사용과 관련된 기타 적용 가능한 약관(각각 수시로 수정 또는 업데이트될 수 있음)도 포함됩니다.

본 서비스 약관을 이해하지 못하거나 이에 구속되기를 원하지 않는 경우 제품에 액세스하거나 사용해서는 안 됩니다. 제품에 액세스, 탐색 또는 다른 방식으로 사용함으로써 귀하는 본 서비스 약관을 읽고 이해했으며 이에 구속되는 데 동의하는 것을 인정합니다. 해당 법률에 따라 계약을 체결할 수 있는 법적 연령은 귀하의 국가("법적 연령")에 따릅니다. 플랫폼을 사용할 수 있는 최소 연령 요건을 충족하지 못하거나 이전에 법적 보호자에 의해 플랫폼에서 정지 또는 제거된 경우 법적 보호자는 플랫폼 약관에 동의해야 합니다. 본 서비스 약관을 수락할 법적 능력이 있는지 여부를 모르는 경우 법적 보호자에게 문의하거나 제품을 사용하거나 플랫폼에 액세스하지 마십시오. 제품을 사용함으로써 귀하는 다음 사항을 진술하고 보증합니다. (A) 귀하가 제출하는 모든 등록 정보는 진실하고 정확합니다. (B) 귀하는 그러한 정보의 정확성을 유지합니다. (C) 귀하의 제품 사용은 적용 가능한 법률 또는 규정을 위반하지 않습니다. (D) 귀하는 이러한 연령 요건을 충족합니다. 귀하의 계정 또는 회사 서비스 사용이 본 서비스 약관을 위반하는 경우 Stripe는 귀하의 신청을 즉시 종료하거나 귀하의 계정을 정지할 수 있습니다.

## 2. 계정

2.1 도구를 사용하려면 플랫폼("계정")과 관련하여 Stripe에 등록된 계정이 필요합니다. 플랫폼을 통해 계정을 생성함으로써 귀하는 다음 사항에 동의합니다. (a) 다른 사람과 비밀번호를 공유하지 않고 계정에 대한 액세스를 제한하여 계정의 보안을 유지합니다. (b) 플랫폼 또는 귀하의 계정과 관련된 보안 침해를 발견하거나 의심되는 경우 Stripe에 즉시 알립니다. (c) 귀하의 계정에서 발생하는 모든 활동에 대한 책임을 지고 무단 액세스의 모든 위험을 감수합니다.

2.2 Stripe는 단독 재량으로 계정 신청을 수락하거나 거부하고/하거나 본 서비스 약관 또는 여기에 참조된 정책을 위반했다고 판단되는 계정을 정지 또는 해지할 수 있습니다. Stripe는 언제든지 어떤 이유로든 귀하에게 사전 통지 없이 그렇게 할 수 있습니다. 계정 활동 및 액세스는 본 서비스 약관의 적용을 받습니다.

2.3 귀하는 항상 귀하의 계정 보안을 유지하고 오용을 방지해야 하며, 귀하의 계정 액세스를 인증하는 데 사용되는 할당된 키 또는 기타 자격 증명과 귀하의 모든 시스템, 장치 및 통신에 대한 액세스를 유지하고 오용을 방지해야 합니다. 귀하는 귀하와 귀하의 활동에 적용되는 PCI 데이터 보안 표준을 준수하고 귀하의 사용자에게 준수하도록 하며, 귀하에게 제공된 귀하의 계정, 비상 키 또는 기타 자격 증명의 보안을 보호하기 위해 카드 협회 및 네트워크 규칙(해당되는 경우)에 따른 적용 가능한 의무를 준수하는 데 동의합니다.

2.4 계정의 각 사용자는 계정을 사용하는 데 제공된 자격 증명의 기밀성 유지 및 적절한 사용에 대해 개별적으로 책임을 집니다. 귀하는 귀하의 계정의 모든 사용과 귀하의 계정을 사용하여 발생하는 모든 행위에 대해 전적으로 책임이 있으며 책임을 집니다. Stripe는 귀하의 계정에 액세스하거나 사용하는 사람이 귀하인지 확인할 의무가 없습니다.

2.5 귀하는 다음 사항을 인정하고 동의합니다. (i) 귀하는 귀하가 그러한 사용을 승인했는지 여부에 관계없이 귀하의 계정을 사용하여 이루어진 플랫폼의 모든 행위 및 사용에 대해 책임을 지고 구속됩니다. (ii) 귀하는 Stripe의 사전 서면 동의 없이 귀하의 계정 권한을 다른 법인에 판매, 양도 또는 서브라이선스하지 않습니다. (iii) 귀하는 귀하의 계정과 관련된 모든 활동에 대한 책임이 있습니다. (iv) 귀하는 귀하의 계정 및 비밀번호의 기밀성을 유지할 책임이 있습니다(시행, 금지, 조례, 규칙, 법률, 규칙, 규정, 통지, 지침, 지침 및 모든 해당 미국 연방, 주 및 지방 정부 및 정부 기관 및 기관의 명령, 그리고 귀하에게 적용될 수 있는 플랫폼에 액세스하는 관할 구역 정부의 법률 및 명령에 따름). Stripe는 귀하의 계정으로 받은 모든 통신이 귀하에 의해 이루어진 것으로 간주할 수 있습니다. 귀하의 계정의 무단 사용을 알게 된 경우 아래 12조에 명시된 이메일 주소로 Stripe에 즉시 알려야 합니다.

## 3. 플랫폼

3.1 플랫폼은 Stripe에서 제공하는 API("API 서비스")를 사용하기 위해 특정 소프트웨어, 애플리케이션 및 서비스("플랫폼")에 액세스하고 사용할 수 있는 기능을 제공합니다.

3.2 귀하는 본 서비스 약관, Shopee 제3자 서비스 약관(https://shopee.sg/legaldoc/terms) 및 애플리케이션 데이터, 사용 정책에 따라 Shopee 서비스를 사용하는 사람들에게 결제 관련 기능을 제공하기 위해 Stripe의 API 서비스와 상호 작용하는 소프트웨어 애플리케이션, 웹사이트 또는 온라인 서비스(각각 "애플리케이션")를 생성, 배포 및 사용하는 데에만 플랫폼 및 도구에 액세스하고 사용할 수 있습니다. 이러한 서비스 약관은 수시로 플랫폼에 게시됩니다(총칭하여 "개발자 약관").

3.3 플랫폼, API 서비스 및/또는 애플리케이션 사용과 관련하여 귀하 및 귀하를 대신하여 행동하는 모든 사람(귀하의 직원, 인력, 하청업체, 대리인, 본인, 계약자 및 하청업체("사용자") 포함(해당되는 경우))은 다음을 수행해야 합니다. (a) 직간접적으로 (i) API 서비스 또는 API 문서(아래 정의됨)의 소스 코드, 객체 코드 또는 기본 구조, 아이디어, 노하우 또는 알고리즘을 리버스 엔지니어링, 리버스 컴파일, 디스어셈블 또는 다른 방식으로 발견하려고 시도하지 않습니다. (ii) API 서비스, 플랫폼 또는 API 문서를 기반으로 수정, 번역 또는 파생물을 생성하지 않습니다. (iii) 다음 방식으로 API 서비스를 사용하지 않습니다. (A) 적용 가능한 법률 또는 규정에 위배됩니다. (B) 타인의 권리(개인 정보 보호 또는 홍보 권리 또는 지적 재산권 포함)를 침해합니다. (C) 유해하고, 사기적이고, 위협적이고, 학대적이고, 괴롭히고, 불법 행위이고, 명예를 훼손하고, 저속하고, 음란하고, 비방적이고, 타인의 개인 정보를 침해하거나 차별, 편협, 인종 차별, 증오, 괴롭힘 또는 해를 조장합니다. (D) 유해하고, 불쾌하거나 부적절한 콘텐츠(불쾌하거나 부적절한 자료("금지 자료") 포함)를 전송, 표시, 배포 또는 다른 방식으로 제공합니다.

3.4 귀하는 귀하에게 부여된 승인 및 액세스 범위를 벗어나는 목적으로 플랫폼 또는 도구에 액세스하거나 사용하지 않거나 본 서비스 약관에 명시된 조건을 위반하는 행위를 하지 않습니다. 특히, 귀하는 본 서비스 약관에 명시적으로 허용된 경우를 제외하고 다음 중 어느 것도 수행하거나 시도하거나 제3자가 수행하도록 승인하거나 허용하지 않습니다. (a) API 서비스 또는 API 문서를 제3자에게 라이선스, 판매, 임대, 리스, 양도, 배포, 게시, 서브라이선스하거나 다른 방식으로 제공하거나 제3자가 이를 사용하도록 허용합니다. 단, 귀하는 귀하의 사용자(아래 정의됨)가 API 서비스를 사용하도록 허용할 수 있습니다. (b) 플랫폼 또는 도구의 무단 복사본을 복제하거나 만듭니다. (c) 플랫폼 또는 도구의 소스 코드를 디컴파일, 디스어셈블, 리버스 엔지니어링하거나 다른 방식으로 파생하려고 시도합니다. (d) 플랫폼 또는 API 서비스의 가용성, 성능 또는 기능을 모니터링하거나 유사한 벤치마킹 목적으로 플랫폼 또는 API 서비스에 액세스하거나 사용합니다. (e) 개인 링크가 포함된 개인 데이터 및 데이터 세트를 포함하여 데이터 사용 정책, 기타 Stripe 약관 또는 정책 또는 적용 가능한 데이터 보호법을 위반하는 정보를 포함하지 않는 영역을 포함하여 개인 데이터를 포함한 데이터를 전송하기 위해 플랫폼, 도구 또는 플랫폼을 통해 또는 플랫폼과 관련하여 전송된 통신을 사용합니다. (f) 지적 재산권의 보안 또는 보호를 위해 플랫폼 또는 도구의 기술적 기능 또는 조치를 우회하거나 비활성화합니다. (g) 플랫폼 또는 도구를 제공하는 데 사용되는 소프트웨어를 해독, 디컴파일, 디스어셈블 또는 리버스 엔지니어링하거나 플랫폼, 도구 또는 플랫폼을 통해 또는 플랫폼과 관련하여 전송된 통신을 기반으로 어떤 식으로든 변경, 수정 또는 파생물을 생성합니다. (h) 로봇, 스파이더, 사이트 검색/검색 애플리케이션 또는 기타 수동 또는 자동 장치 또는 프로세스를 사용하여 플랫폼의 탐색 구조 또는 프레젠테이션을 검색, 인덱싱, "스크랩", "데이터 마이닝"하거나 어떤 식으로든 복제하거나 우회합니다. (i) Stripe의 이름, 상표 또는 제품 이름을 활용하는 메타 태그 또는 기타 "숨겨진 텍스트"를 사용합니다. (j) 플랫폼을 손상시키거나, 비활성화하거나, 과부하를 주거나, 손상시킬 수 있는 기타 활동을 수행합니다. (k) 사용자의 동의 없이 사용자 데이터를 수집하거나 수집합니다. (l) 서비스 거부 유형 공격을 통해 시스템 또는 네트워크의 취약점을 조사, 스캔 또는 테스트하거나 보안 또는 인증 조치를 위반하려는 시도를 포함하여 플랫폼 또는 Stripe의 시스템 또는 네트워크에 과부하를 주거나 손상시키는 방식으로 애플리케이션, 플랫폼 및 도구를 사용합니다. ## 10. 수정 사항

10.1 Shopee는 이유나 사전 통지 없이 수시로 귀하에 대한 책임을 지지 않고 플랫폼, 도구, Shopee 서비스, Shopee 콘텐츠 또는 플랫폼 및 도구의 다른 부분 또는 기능에 대한 사용 또는 액세스를 수정, 업그레이드, 일시 중단, 중단하거나 제한을 둘 수 있습니다. 귀하의 애플리케이션 또는 프로토콜에 영향을 줄 수 있는 해당 수정, 업그레이드 및 변경은 도구에 액세스하지 못할 수 있습니다. 수정 사항이 귀하의 애플리케이션에 영향을 미치는 경우 Shopee는 해당 수정 사항을 귀하에게 통지하기 위해 합리적인 노력을 기울여야 합니다. 이에 따라 귀하의 애플리케이션의 플랫폼 및 도구 사용을 방해하거나 간섭합니다 ("수정").

10.2 Shopee는 플랫폼을 통해 도구에 대한 통지 또는 Shopee가 지정할 수 있는 다른 통지 방법 (특정 사용자 또는 모든 사용자에게 이메일 통지를 포함해야 함)을 통해 도구, 도구 또는 콘텐츠에 대한 업데이트가 사용자에게 영향을 미치는 경우 플랫폼 도구, Shopee 콘텐츠 또는 Shopee 서비스의 전부 또는 일부 제공.

10.3 귀하는 Shopee가 (i) 플랫폼 또는 도구 사용 등록과 관련하여 언제든지 귀하에게 통신을 보내거나 통지할 수 있음을 인정합니다. 해당 날짜 이후에는 해당 변형을 수락한 것으로 간주됩니다. 변형을 수락하지 않으면 플랫폼 또는 도구 사용을 중단하고 Shopee에 즉시 수락하지 않음을 통지해야 합니다. Shopee에 통지하지 않으면 플랫폼 또는 도구에 대한 지속적인 액세스 또는 사용은 변경된 조건을 수락한 것으로 간주됩니다.
