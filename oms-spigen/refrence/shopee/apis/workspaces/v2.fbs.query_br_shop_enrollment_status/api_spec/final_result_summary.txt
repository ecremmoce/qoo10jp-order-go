Endpoint: HTTP /v2/fcx/query_br_shop_enrollment_status
Description: This API checks whether a given shop_id is eligible to enroll in the Brazil Fulfiled by Shopee (FBS) service

Parameters (5):
----------------------------------------------------------------------
* partner_id (int)
   Partner ID is assigned upon registration is successful. Required for all requests.
* timestamp (timestamp)
   This is to indicate the timestamp of the request. Required for all requests. Expires in 5 minutes
* access_token (string)
   The token for API access, using to identify your permission to call APIs. Valid for multiple use and expires in 4 hours.
* shop_id (int)
   Shopee's unique identifier for a shop. Required param for most APIs
* sign (string)
   Signature generated by partner_id, api_path, timestamp, access_token, shop_id, shop_enrollment_status_key via HMAC-SHA256 hashing algorithm.

Response Fields (6):
----------------------------------------------------------------------
  message (string)
   Indicate error details if ret error. Empty if no error happened.
  error (string)
   Indicate error type if ret error. Empty if no error happened.
  shop_id (int64)
   Shopee's unique identifier for a shop
  enrollment_status (int32)
   1: enable enrollment
2: disable enrollment
3: already enrollment
  enable_enrollment_time (int64)
   The time of this shop able to enroll FBS
  request_id (string)
   The identifier for an API request for error tracking
