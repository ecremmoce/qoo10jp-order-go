# 상품정보고시 코드 조회
**분류**: reference

## 개요

- 상품정보고시 항목 등록을 위해 제공하는 API 입니다.

- 상품관리 > 사방넷상품속성관리 - 도움자료 에서 속성코드 전체 리스트를 다운 받을 수 있습니다.

- 웹에서 접근 가능한 URL 또는 IP 주소가 존재 해야 합니다.

## 사방넷 원본 엔드포인트

```
* https://sbadminXX.sabangnet.co.kr/RTL_API/xml_goods_prop_code_info.html?xml_url=상품_속성코드_조회_xml주소
```

## RESTful API 변환

**Method**: `POST`

**Path**: `/api/v1/상품정보고시 코드 조회`

**Summary**: 상품정보고시 코드 조회

## 데이터 필드 (Data Fields)

| XML 필드명 | JSON 필드명 | 설명 | 필수 | 비고 |
|-----------|-------------|------|------|------|
| PROP1_CD | prop1Cd | 속성분류코드 | Y | 상품 속성분류코드를 입력합니다. 예 : 의류는 001을 입력합니다. |

## XML 구조 예제

```xml
<SEND_COMPAYNY_ID>
  <HEADER>
  </HEADER>
  <DATA>
    <ITEM>
      <PROP1_CD>값</PROP1_CD>
    </ITEM>
  </DATA>
</SEND_COMPAYNY_ID>
```

## JSON 구조 예제

```json
{
  "prop1Cd": "값"
}
```
