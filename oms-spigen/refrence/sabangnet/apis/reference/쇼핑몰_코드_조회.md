# 쇼핑몰 코드 조회
**분류**: reference

## 개요

- 사방넷 메뉴 기본정보 >쇼핑몰관리(지원) 에 세팅 되어 있는 거래중인 쇼핑몰을 조회 할 수 있습니다.

- 웹에서 접근 가능한 URL 또는 IP 주소가 존재 해야 합니다.
