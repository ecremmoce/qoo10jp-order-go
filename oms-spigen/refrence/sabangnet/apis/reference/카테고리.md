# 카테고리
**분류**: reference

## 개요

- 사방넷 메뉴 기본정보 >카테고리관리 에 등록된 카테고리를 조회하는 API 입니다.

- 카테고리 조회 API에서 조회된 코드를 상품에 등록하여 사용합니다.

- 웹에서 접근 가능한 URL 또는 IP 주소가 존재 해야 합니다.

## 사방넷 원본 엔드포인트

```
* https://sbadminXX.sabangnet.co.kr/RTL_API/xml_category_info.html?xml_url=카테고리조회xml주소
```

## RESTful API 변환

**Method**: `POST`

**Path**: `/api/v1/카테고리`

**Summary**: 카테고리

## 인증 (Authentication)

### 사방넷 XML 헤더

| 필드명 | 설명 | 필수 |
|--------|------|------|
| SEND_COMPAYNY_ID | 사방넷 로그인아이디 | Y |
| SEND_AUTH_KEY | 사방넷에서 발급한 인증키 | Y |

### RESTful HTTP 헤더

```http
Authorization: Bearer {API_KEY}
X-Company-ID: {COMPANY_ID}
Content-Type: application/json
```

## 헤더 필드 (Header Fields)

| 필드명 | 설명 | 필수 | 비고 |
|--------|------|------|------|
| SEND_COMPAYNY_ID | 사방넷 로그인아이디 | Y | 사방넷 어드민 로그인 아이디입력 |
| SEND_AUTH_KEY | 사방넷에서 발급한 인증키 | Y | 사방넷에서 발급 받은 인증키(만약 받지 않았다면 사방넷에서 인증키 발급 요청) |
| SEND_DATE | 전송일자 | Y | YYYYMMDD |
